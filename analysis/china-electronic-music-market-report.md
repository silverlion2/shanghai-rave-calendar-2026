### 2.2 价值流向分析

以一场 **地下Club电子活动（200人规模）** 为例：

| 参与方 | 收入/支出 | 占总消费比 | 说明 |
|--------|----------|-----------|------|
| **Club/场地** | ¥40,000-60,000 | 50-60% | 酒水+场地费，主要利润方 |
| **主办方/活动方** | ¥15,000-25,000 | 20-30% | 门票收入（扣除DJ费用） |
| **DJ（国际/知名）** | ¥5,000-30,000+ | 10-30% | 费用差异极大，国际艺人可达¥5万+ |
| **DJ（本地）** | ¥1,000-5,000 | 2-8% | 本地DJ收入偏低 |
| **票务平台** | ¥1,000-3,000 | 3-5% | 抽成8-15% |
| **营销/宣传** | ¥3,000-8,000 | 5-10% | 小红书投放、KOL合作 |
| **用户端总消费** | ¥80,000-100,000 | 100% | 门票(¥100-200) × 200人 + 酒水 |

**关键发现**：门票仅占用户总消费的30-50%，**酒水才是真正大头**，但酒水完全被场地截获，信息分发方无法参与分润。

### 2.3 产业链痛点

| 痛点 | 影响方 | 严重程度 | 现状 |
|------|--------|---------|------|
| 信息极度碎片化 | 用户 | ⭐⭐⭐⭐⭐ | 散落在小红书、微信、场地官网、朋友圈 |
| 本地DJ收入偏低 | 创作者 | ⭐⭐⭐⭐ | 严重依赖演出排期，收入不稳定 |
| 中小型活动营销成本高 | 主办方 | ⭐⭐⭐⭐ | 缺乏精准投放渠道，ROI低 |
| 票务平台抽成高 | 主办方/用户 | ⭐⭐⭐ | 8-15%抽成 |
| 国际艺人信息获取难 | 全链 | ⭐⭐⭐ | 主要依赖RA和境外渠道 |
| 版权监管不确定性 | 全链 | ⭐⭐ | 地下活动常有灰色空间 |

### 3.2 人群核心特征

| 维度 | 描述 |
|------|------|
| **年龄分布** | 主力 19-32岁，DJ/制作人圈层可延至40+ |
| **性别比例** | 地下活动约55%男 / 45%女；商业EDM夜店女性略高 |
| **城市线级** | 一线+新一线占比>85%；上海是全国最活跃市场 |
| **收入水平** | 中等偏上（¥8000-30000/月），但学生党占20-30%（由父母/兼职支撑） |
| **职业分布** | 创意行业（设计/艺术/媒体/自由职业）+ 留学生 + 金融/IT白领 |
| **教育水平** | 本科及以上占比>70% |
| **海外背景** | 约30-50%有海外留学/生活经历（影响音乐品味形成） |
| **消费倾向** | 体验型消费优先于物质消费；对"氛围感"和"阵容"极度敏感 |

### 3.3 文化身份与PLUR

- **核心文化信条**：PLUR (Peace, Love, Unity, Respect)
- **中国化演变**：从反主流亚文化 → 时尚生活方式符号
- **鄙视链**：真·地下 Techno 粉 > House 独立圈 > 商业 EDM 粉 > 打卡消费者
- **社群归属感强**：以场地、厂牌、DJ 为核心形成紧密社群

### 3.4 消费决策链

```
① 信息发现（小红书/朋友圈/微信群/场地公众号/DJ账号/RA）
         ↓ 被动刷到 or 主动搜索
② 评估（DJ 阵容 > 场地口碑 > 票价 > 时间/日期）
         ↓ 冲动型 vs 计划型
③ 购票（秀动/大麦/微信群预售/现场票/小程序）
         ↓
④ 到场（周五/周六 22:00-23:00 进场高峰）
         ↓
⑤ 二次消费（酒水¥100-250，可能更高）
         ↓
⑥ 社交传播（小红书打卡/微信分享/推荐朋友，形成新的①）
```

**关键洞察**：①信息发现环节是整条链中**最混乱、最分散、体验最差**的环节 —— 这是你的项目切入的最佳位置。

### 3.5 生命周期与LTV

| 阶段 | 时长 | 特征 | 年消费 | LTV贡献 |
|------|------|------|--------|---------|
| 入门期 | 0-1年 | 被朋友带入，广泛尝试 | ¥5000-15000 | 低 |
| **活跃期** | **1-5年** | **形成稳定偏好，高频消费** | **¥18000-45000** | **核心贡献** |
| 沉淀期 | 5年+ | 选择性参与，成为意见领袖 | ¥10000-30000 | KOL价值 |
| 退出 | 工作/家庭/健康原因 | 但仍关注圈内资讯 | — | 信息消费者 |

**核心用户LTV估算**：活跃期3-5年 × 年消费¥1.8-4.5万 = **¥5.4万 - ¥22.5万/人**

### 3.6 与普通年轻人消费对比

## 第四部分：竞品分析

### 4.1 全球/国内主要玩家

| 产品 | 定位 | 覆盖 | 优势 | 劣势 |
|------|------|------|------|------|
| **Resident Advisor (RA)** | 全球电子音乐门户 | 全球，中国仅一线/部分二线 | 最权威、DJ档案最全、国际活动覆盖广 | 英文为主、不覆盖地下Club、Cloudflare反爬阻断数据获取 |
| **Songkick** | 音乐演出发现/票务 | 全球，以主流演出为主 | 与Bandsintown整合，数据全 | 非电音垂直、中国活动覆盖极有限 |
| **Bandsintown** | 演出发现 + 购票 | 全球，主流音乐 | 追踪艺人功能强 | 同上，非垂直 |
| **Raveland APP** | 中国电音垂直工具 | 中国 | 定位明确，百大夜店收录 | 信息更新慢，偏商业夜店，2021年后活跃度存疑，非地下圈偏好 |
| **秀动 ShowStart** | 独立音乐票务平台 | 全国，以独立音乐/Livehouse为主 | 票务生态完整，太合音乐集团背景 | 非电子音乐垂直，信息筛选困难 |
| **大麦/猫眼** | 综合票务 | 全国，大型活动为主 | 市场份额大 | 地下小活动几乎不覆盖 |
| **小红书** | 社交+信息 | 中国，实际是raver最大信息来源 | 图文传播力强 | 信息非结构化，搜索效率低 |
| **微信公众号/群** | 私域信息 | 分散 | 即时、精准 | 不聚合，信息孤岛 |

### 4.2 竞争格局分析

```
                   信息完整度
                     ↑
      垂直深度      │      RA (国际权威)
         ↑          │
         │          │
   ✅  你的项目     │        Raveland（偏商业）
        定位区间    │      秀动（偏独立音乐）
         │          │
         │          │   小红书（信息流但非结构化）
         └──────────────────────────────────→  垂直程度（电音专属）

   【机会】：高度垂直 + 中文 + 精准地下场景 = 几乎无直接竞品
```

**结论**：地下电子音乐垂直信息工具在中文市场是**空白地带**，RA因语言和覆盖深度不够、Raveland偏商业夜店、秀动偏独立音乐。你的项目具备先入优势。

### 5.4 风险提示

| 风险 | 说明 | 缓解方案 |
|------|------|---------|
| 市场规模天花板 | 地下电音是小众市场 | 向"全电子音乐"扩展，+ 跨城市扩张 |
| 内容获取难度 | 活动信息分散，爬取成本高 | 众包 + 合作协议 + AI自动抓取 |
| 商业夜店 vs 地下圈矛盾 | 商业合作可能影响地下圈认可度 | 保持编辑独立性，区分"推荐"与"广告" |
| 政策风险 | 夜间经济/音乐活动监管不确定性 | 合规运营，聚焦信息聚合而非票务 |

## 第六部分：项目建议（Actionable Insights）

基于以上分析，对 rave calendar 的产品和战略建议：

1. **立即强化**：上海地下电子音乐活动收录深度，做到"只要有活动就能搜到"——这是核心护城河
2. **用户增长**：利用现有DJ社群和场地KOL冷启动，而非付费广告 —— raver之间的病毒传播系数很高
3. **信息结构化**：将小红书的图文信息转化为结构化数据（日期/时间/场地/DJ阵容/曲风/票价/海报），这是比小红书更强的搜索体验
4. **国际化补充**：RA的国际艺人行程是重要补充，可以建立镜像/本地化版本
5. **数据产品化**：随着用户量积累，你的活动数据可以成为行业研究工具，卖给主办方/品牌方做市场分析

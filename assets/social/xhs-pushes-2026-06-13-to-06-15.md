## 整体口吻规范

小红书风格：

```text
短句。
强 hook。
像朋友发来的今晚选择，不像正式媒体通稿。
多用“适合谁”“怎么选”“别一场场翻”的表达。
每条正文前 3 行必须让人知道：这篇能帮我决定去哪。
```

Techno 酷炫措辞方向：

```text
硬一点 / 快一点 / 地下室能量 / 低频压住
工业冷感 / 机器感 / 合成器冷光 / 高 BPM
暗场 / 冷房间 / 声压 / trance 提速
不想太乖 / 不走游客夜生活 / 别只去人挤人的酒吧
先 rooftop 热身，再进黑房间
```

边界：

```text
不写违法暗示。
不暗示药物、失控、未成年夜生活。
不把 watch / 单一来源信息写成百分百确认。
每条结尾保留“出发前看官方/票务页确认”。
```

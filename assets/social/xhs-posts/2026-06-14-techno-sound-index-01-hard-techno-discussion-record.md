## Core Direction

We decided to add a Xiaohongshu content line that introduces specific techno subgenres one post at a time.

The first post introduces only `Hard Techno`.

Series rule:

```text
一次只介绍一个声音。
不要在一篇里同时讲太多 subgenre。
```

Editorial promise:

```text
让第一次接触 techno 的人能听懂，
也让已经听过一点的人觉得判断标准靠谱。
```

## Tone

The voice should be slightly literary but immediately translated into plain language.

Pattern:

```text
文一点：
Hard Techno 像一台没有耐心的机器，把鼓点推到你胸口。

说人话：
鼓更重、更靠前，房间压力更强，不太适合一直聊天。
```

Tone boundary:

```text
可以有画面感。
不要玄学。
不要写成夜店宣传。
不要用药物、失控、危险或非法派对暗示。
```

## Research Takeaways

The Hard Techno explanation borrowed listener-facing cues from public genre explainers and production tutorials.

Useful concepts:

```text
distorted kick -> 鼓更粗、更硬、更靠前
rumble kick -> kick 后面拖着一条低频尾巴
sidechain / pumping -> kick 和低频之间有挤压感
saturation / overdrive -> 声音更糙、更热、更有磨损感
industrial texture -> 机器感、金属感、冷硬纹理
abrupt drop -> 变化来得更直接，不是慢慢铺开
```

Editorial translation:

```text
Hard Techno 不只是快。
它更像更直接的 kick、更重的声压、更少的松弛空间，
以及更高强度、更集中的房间状态。
```

## Soundtrack / QR Compliance

Decision:

```text
Do not put QR codes, external links, music-platform logos, playlist URLs, or "scan to listen" instructions inside the Xiaohongshu images.
```

Avoid:

```text
扫码听歌单
完整歌单在某平台
主页链接自取
评论歌单私信发链接
去 Spotify / 网易云 / SoundCloud 搜我的 playlist
```

Lower-risk alternatives:

```text
Use artist / DJ names as text coordinates.
Use Xiaohongshu's native BGM if making a video version.
If relevant public Xiaohongshu accounts exist, @ only verified/public accounts after manual checking.
Do not use @ as disguised off-platform traffic guidance.
```

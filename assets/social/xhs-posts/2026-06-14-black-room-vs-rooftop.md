## 正文

```text
前两篇我们都在给答案：

6.12-6.13 周末清单。
6.13 今晚快速版。

但选电子音乐活动，不是只看“哪场最火”。
第一步其实很简单：

你今晚要进黑房间，
还是先 rooftop 热身？

这是 Basement Dispatch 选场指南 01。
这篇只讲一个判断题。

01 什么是黑房间？

你可以理解成：
灯光更暗，声音更集中，聊天欲望更低。

关键词通常是：
hard techno / techno / trance / EBM / industrial / high BPM

更适合：
想听 kick、声压、低频、机器感的人。
想把注意力交给声音，不太想社交的人。
想要“今晚被节奏推着走”的人。

可能不适合：
第一次约会。
想边喝边聊。
对高音量、高密度鼓点比较敏感。

02 什么是 rooftop 热身？

你可以理解成：
先把夜晚打开，但不立刻进入高压状态。

关键词通常是：
rooftop / Bund view / house / DJ sets / sunset session

更适合：
朋友集合。
约会友好。
想先看风景、喝一杯、进入状态。
不确定今晚要不要继续去 club 的人。

可能不适合：
只想要重型 techno。
想要黑房间和持续声压。
讨厌社交场景的人。

03 怎么快速判断？

如果你今晚想：
少说话 / 高 BPM / 低频压住 / 机器感
那就往黑房间看。

如果你今晚想：
先见朋友 / 看风景 / 轻一点热身 / 后面再决定
那就先 rooftop。

04 用昨晚的例子看

Abyss / EXIT 这一类更接近黑房间逻辑：
更暗、更集中、更靠声音往前推。

Pop Bar / Flair 这一类更接近 rooftop 逻辑：
更开放、更社交、更适合把夜晚慢慢打开。

没有绝对好坏。
只有你今晚适合哪种能量。

之后每周清单，我们都会继续按这个逻辑拆：
黑房间 / rooftop / club night / dark wave / machine sound

完整日历看主页。

评论区留：
「黑房间」= 想看上海黑房间入门
「rooftop」= 想看 rooftop 热身清单
「场地」= 想看场地速查

出发前请再看官方/票务页面确认时间、票价和阵容。
```

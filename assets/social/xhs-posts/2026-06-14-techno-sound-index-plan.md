# TECHNO SOUND INDEX - Xiaohongshu Content Plan

Updated: 2026-06-14 Asia/Shanghai

## Positioning

`TECHNO SOUND INDEX` is a Xiaohongshu content line for Basement Dispatch.

Each post introduces one specific techno-related sound. The goal is not to build an encyclopedia. The goal is to help readers judge whether a Shanghai electronic-music event fits them before they leave home.

Core question:

```text
看到活动海报里的 hard techno / hypnotic / industrial / acid 这些词，我该怎么判断这场适不适合我？
```

## Editorial Rules

- One post explains one sound only.
- Use listening cues, room cues, and audience fit instead of music-theory lectures.
- Keep claims practical: what it sounds like, who it is for, who may not enjoy it, and how to read an event listing.
- Do not imply drug use, loss of control, illegal parties, or unsafe behavior.
- Do not claim an event is confirmed unless the current calendar source confirms it.
- Do not say a venue is always one genre. Say: check the lineup, promoter, ticket page, and current poster.
- Avoid empty praise such as "best", "top", "legendary", "must-go", or "ceiling".

## First Batch

### 01 - Hard Techno

Angle:

```text
Hard Techno 是什么？不是所有快歌都叫 hard
```

Use first because it has high search value and high misunderstanding risk. It also sets the account's judgment standard: intensity alone is not enough; readers should check sound, room, lineup, and source.

### 02 - Hypnotic Techno

Angle:

```text
Hypnotic Techno 是什么？不是助眠，是循环把你带进去
```

Use to build taste depth after the high-intensity first post.

### 03 - Industrial Techno

Angle:

```text
Industrial Techno 是什么？冷、硬、机器感，不等于普通黑暗
```

Useful for connecting techno with EBM, darkwave, and machine-sound events without collapsing them into one label.

### 04 - Acid Techno

Angle:

```text
Acid Techno 是什么？听到 303 那条酸线就懂了
```

Keep the technical term light. Explain it as a recognizable squelchy synth line, not as a gear lecture.

### 05 - Hardgroove

Angle:

```text
Hardgroove 和 Hard Techno 差在哪？一个更弹，一个更冲
```

Good follow-up when comments start asking why some fast techno still feels more danceable.

### 06 - Minimal Techno

Angle:

```text
Minimal Techno 是什么？少不等于无聊
```

Use to expand beyond hard-rave expectations.

### 07 - Dub Techno

Angle:

```text
Dub Techno 是什么？更深、更湿、更有空间感
```

Best for listening-room or lower-pressure entries.

### 08 - Detroit / Groovy Techno

Angle:

```text
Detroit / Groovy Techno 是什么？机器感里也可以有弹性
```

Use after the audience already understands that techno is not only hard and dark.

### 09 - Melodic Techno

Angle:

```text
Melodic Techno 是什么？有旋律，但不一定适合黑房间
```

Useful for correcting the "好听旋律 = techno 主场" assumption.

### 10 - EBM and Techno

Angle:

```text
EBM 为什么经常和 Techno 放在一起？
```

Treat this as adjacent context, not pure techno.

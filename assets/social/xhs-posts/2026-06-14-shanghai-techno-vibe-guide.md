```text
前两篇我们做了两件事：

第一篇：6.12-6.13 周末清单。
第二篇：6.13 今晚快速版。

都是直接给答案：

但很多人真正卡住的不是“有没有活动”，而是：

这场到底适不适合我？

所以开一个新系列：
Basement Dispatch 选场指南 01

先讲最基础的 5 种 vibe。
下次看到活动名、场地名、genre，你就能大概判断是不是你的房间。

简单说：
前两条你照着选。
这一条你学会自己选。

收藏这篇。
以后看周末清单会快很多。

01 硬、快、强度高
关键词：
Abyss / hard techno / trance / new trance / 高 BPM

适合：
想要提速、声压、黑房间和更直接的 kick。
如果你今晚不想聊天，只想被节奏推着走，看这一类。

02 稳一点的 club night
关键词：
POTENT / Reactor / Dirty House / techno / house / bass / UKG

适合：
想跳舞，但不想整晚高压。
这种更像“可以进状态，也可以社交”的夜晚。

03 dark / wave / EBM
关键词：
Specters / Love Bang / darkwave / EBM / goth / industrial

适合：
喜欢暗色、冷感、酒吧气质的人。
不是游客夜生活，更像本地地下小房间。

04 machine / electro / industrial
关键词：
EXIT / electro / IDM / industrial / live set / 合成器冷光

适合：
喜欢机械感、鼓机、合成器和更锋利声音的人。
这类不一定“嗨”，但很有质感。

05 rooftop / 约会友好
关键词：
Pop Bar / Flair / rooftop / Bund view / house / DJ sets

适合：
想先喝一杯、看风景、热身。
适合朋友集合，也适合不想一上来就进黑房间。

之后每周的周末清单，我们都会按这个逻辑拆：
硬一点 / 稳一点 / 暗一点 / 机器感 / rooftop 热身

完整日历看主页。

评论区留：
「今晚」= 看当天选择
「周末」= 看完整周末清单
「场地」= 看上海电子音乐场地速查

出发前请再看官方/票务页面确认时间、票价和阵容。
```

## Editorial Intent

This post should help readers decide whether a hard-techno-leaning night fits them.

It should not become a full techno history post, a playlist article, or a generic nightlife recommendation.

Core message:

```text
Hard Techno 的重点不是“快”一个字，而是更直接的 kick、更高的声压、更少的松弛空间，和更集中、更高强度的房间状态。
```

## Tone Direction

Use a two-layer voice:

```text
First layer:
有一点文气，有画面感，适合封面、开头、转场。

Second layer:
马上翻成白话，让第一次接触 techno 的人也能懂。
```

Rule:

```text
每一句文一点的表达，后面都要能接一句“说人话就是……”。
```

Example:

```text

说人话：
鼓更重、更靠前，房间压力更强，不太适合一直聊天。
```

```text
1. Hard Techno 不只是快。

5. 在上海选场时，要看当天阵容和主办，不要只看场地名或 techno 这个词。
```

## Online Research Notes

Checked public genre explainers and production tutorials on 2026-06-14.

Useful patterns to borrow:

```text
1. Tempo:
Many explainers place Hard Techno around 140-160 BPM, but this should be presented as "often faster", not as a strict rule.

2. Drum core:
Descriptions and tutorials repeatedly center the sound on heavy / distorted kicks.

3. Low-end:
Production tutorials often teach rumble kick or rumbling low-end: a kick tail / low-frequency body that keeps pushing after the transient.

4. Processing:
Tutorial language often includes distortion, saturation, overdrive, compression, limiting, and sidechain.

5. Texture:
Common description words include raw, aggressive, industrial, darker, acid-influenced, hardstyle-influenced, screech synths, and abrupt drops.

6. Arrangement:
Compared with slower, long-build techno, Hard Techno is often described as more immediate, with faster changes, bigger drops, and less gradual unfolding.
```

Audience-friendly translation:

```text
distorted kick -> kick 更粗、更硬、更靠前
rumble kick -> kick 后面拖着一条低频尾巴
sidechain / pumping -> kick 和低频之间有明显挤压感
saturation / overdrive -> 声音更糙、更热、更有磨损感
industrial texture -> 机器感、金属感、冷硬纹理
abrupt drop -> 不慢慢铺，变化来得更突然
```

Use these as editorial language, not as production claims about every track.

### Page 03 - Kick / Rumble Glossary

```text
判断点 01/05 | Kick / Rumble

Kick / Rumble 怎么听？

先听鼓的“头”和“尾巴”。

Distorted kick：失真鼓。鼓更粗、更硬、更靠前。
Transient：鼓刚敲下去的头一下，决定冲击感。
Rumble：kick 后面的低频尾巴。
Tail：尾音。越长越容易有持续压迫感。
Low-end：低频区域，很多时候身体先感觉到。
Sub：更低的低频，未必大声但很顶。

说人话：
不是只听“咚”一声，还要听咚后面有没有往前滚。
```

### Page 04 - Processing / Texture Glossary

```text
判断点 02/05 | Processing / Texture

处理和质感

这些词不是玄学，是听感线索。

Saturation：饱和。让声音更热、更厚。
Overdrive：过载。让声音更糙、更有磨损感。
Compression：压缩。把鼓和低频压得更紧。
Sidechain：侧链。kick 进来时，低频让开一下。
Industrial texture：工业质感。机器、金属、冷硬纹理。
Acid line：酸性线条，常见 303 式扭动感。
Hardstyle-influenced：受 hardstyle 影响，kick 更夸张、更外放。

说人话：
它听起来更粗、更热、更紧，不像干净圆润的背景音乐。
```

## Soundtrack / Playlist Compliance

Question:

```text
能不能在核实页或最后一页加二维码，引导到 soundtrack / playlist？
```

Short answer:

```text
不建议。
在小红书笔记图片里放二维码，引导用户去站外音乐平台、网页、播放列表或私域，属于高风险导流。
```

Why:

```text
小红书公开社区/品牌号规范里，导流相关风险通常包括网址链接、二维码、水印、其他平台信息等。
尤其是图片里的二维码，平台 OCR 和审核很容易识别。
即使不是卖票或交易，只要明显把用户带离小红书，也可能影响推荐或触发审核。
```

Do not use:

```text
二维码
短链
外部 playlist URL
图片角落水印式网址
“扫这里听”
“去 Spotify / 网易云 / SoundCloud 搜我的歌单”
评论区暗号导流
私信发链接
```

Lower-risk alternatives:

```text
1. 只给 artist / track name 文本坐标
例如：
```

## Comment Reply Bank

Use these replies if comments ask follow-up questions.

### If someone asks "新手能去吗？"

```text
能去，但不一定建议从最硬的场开始。先看当天阵容、时间和场地房间；如果你能接受高音量、密集鼓点和少聊天的状态，再考虑 hard techno 方向。
```

### If someone asks "是不是越快越好？"

```text
不是。Hard Techno 不是速度比赛。更关键的是 kick 是否直接、声压是否重、推进是否集中，以及你今晚是否想要这种高强度房间。
```

### If someone asks "rumble 是什么？"

```text
可以理解成 kick 后面拖着的一条低频尾巴。它会让房间有持续往前压的感觉，但不是每一首 Hard Techno 都必须用同一种 rumble。
```

### If someone asks "distorted kick 是什么？"

```text
就是更粗、更硬、更有磨损感的鼓，不是干净圆润的 kick。听感上会更靠前，也更有冲击力。
```

### If someone asks "上海哪个场地是 hard techno？"

```text
不要把场地固定等同于一个曲风。上海很多场地会按当天主办和 lineup 变化。看场地名只能作为线索，真正要看当天海报、阵容和票务/官方页面。
```

### If someone asks "适合约会吗？"

```text
如果对方也喜欢高强度电子音乐，可以。如果是第一次约会、想边聊边熟悉，Hard Techno 通常不是最轻松的选择。
```

### If someone asks "有没有歌单？"

```text
这个系列先帮大家看懂活动和房间状态。歌单可以单独做一期，但不要只靠歌单判断一场活动，现场还是看当天 DJ、主办和房间。
```

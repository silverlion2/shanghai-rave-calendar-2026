# Shanghai Rave Calendar 2026

Interactive static calendar for Shanghai techno, rave, warehouse, industrial, bass, and underground electronic events in 2026.

## Local preview

Open `index.html` directly in a browser, or run:

```bash
npm run check
npx serve .
```

## Deploy

The project is static and can be deployed on Vercel with no build step:

```bash
vercel --prod
```
